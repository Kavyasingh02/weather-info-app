# 🌦️ React Weather App

A simple and modern weather application built with **React.js** that allows users to search for any city and view its current weather conditions in real time using the **OpenWeatherMap API**.

---

## 🚀 Features

- 🔍 Search weather by city name  
- 🌡️ Displays:
  - Current temperature  
  - Minimum & maximum temperature  
  - Feels like temperature  
  - Humidity  
  - Weather description  
- ⚡ Real-time data from OpenWeatherMap API  
- 🎨 Clean UI using Material UI  
- ❌ Error handling for invalid city names  

---

## 🛠️ Tech Stack

- React.js  
- JavaScript (ES6)  
- Material UI  
- OpenWeatherMap API  
- CSS
